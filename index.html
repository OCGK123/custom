당신만의 커스텀 명령어를 만들어보세요!

커스텀봇은, 서버마다 다른 커스텀 명령어를 코딩 없이 쉽게 만들 수 있는 디스코드 봇입니다. 텍스트 응답, 역할 부여, 임베드 등 다양한 명령어를 관리자 권한 없이도 간편하게 설정하세요!

간편 설정

복잡한 설정, 코딩 없이 빠르게 명령어를 생성할 수 있어요

서버별 설정

각 서버마다 독립적인 명령어를 관리할 수 있어요

고급 기능

조건부 랜덤 응답 카운터 등을 코딩없이 쉽게 써보세요

다양한 기능

커스텀봇은 단순한 명령어 봇이 아니라 직접 만들어가는 봇이에요!

텍스트 명령어

간단한 텍스트 응답 명령어부터 다양한 플레이스홀더를 활용한 동적 응답까지, 원하는 대로 설정할 수 있어요!

역할 명령어

명령어 한 번으로 역할을 부여하거나 제거할 수 있어요 토글 방식과 일회성 방식 모두 지원해요

임베드 명령어

색상, 이미지, 필드를 포함한 풍부한 임베드 메시지를 손쉽게 만들 수 있어요!

랜덤 응답

여러 응답 중 하나를 무작위로 선택하는 명령어로 재미있는 요소를 추가해보세요!

조건부 명령어

역할, 채널, 시간대, 요일, 카운터 값에 따라 다른 응답을 제공하는 고급 조건부 명령어를 코딩 없이 만들 수 있어요!

카운터 시스템

유저별 카운터를 생성하고 관리할 수 있어요, 포인트, 경험치 등 다양한 용도로 활용해보세요!

타이머 기능

지정된 시간 후에 메시지를 보내는 타이머를 설정할 수 있어요!

서버 변수

서버 내에서 사용할 수 있는 변수를 저장하고 불러와 명령어에 활용할 수 있어요!

명령어 권한

특정 역할만 사용할 수 있는 명령어를 설정하여 서버 관리를 효율적으로 할 수 있어요!

다양한 명령어

커스텀봇은 다양한 종류의 명령어를 사용할 수 있어요. 서버에 맞게 자유롭게 커스터마이징 해보세요!

일반 명령어 만들기

/일반만들기

텍스트 응답 명령어를 만들어요. 플레이스홀더를 사용하여 동적인 응답을 만들 수 있어요

/일반만들기 명령어:인사 응답:안녕하세요, %useduser%님! %server% 서버에 오신 것을 환영합니다!

명령어 검색

/검색

서버에 등록된 명령어를 검색해요 특정 키워드로 검색하여 빠르게 명령어를 찾을 수 있어요!

/검색 검색어:인사

명령어 목록

/목록

서버에 등록된 모든 명령어 목록을 확인해요. 종류별로 필터링하여 볼 수도 있어요!

/목록 종류:모두

역할 명령어 만들기

/역할만들기

역할 부여 명령어를 만들어요. Toggle 모드에서는 역할을 추가/제거하고, Once 모드에서는 역할이 없을 때만 추가해요!

/역할만들기 명령어:게임 역할:@게이머 방식:Toggle

권한 설정

/권한설정

명령어 사용 권한을 특정 역할로 제한할 수 있어요! 서버 관리를 더욱 효율적으로 할 수 있어요.

/권한설정 명령어:게임 역할:@모더레이터

임베드 명령어 만들기

/임베드만들기

임베드 출력 명령어를 생성합니다. 제목, 설명, 색상 등을 설정할 수 있어요!

/임베드만들기 명령어:공지 제목:서버 공지사항 부제목:중요한 내용을 확인해주세요!

임베드 편집

/임베드편집

임베드 명령어를 상세하게 편집할 수 있어요. 필드 추가, 색상 변경, 이미지 추가 등 다양한 편집이 가능해요!

/임베드편집 명령어:공지 편집모드:필드 추가

카운터 만들기

/카운터만들기

유저별 카운터를 만들어요, 포인트, 경험치 등 다양한 용도로 활용할 수 있어요!

/카운터만들기 카운터이름:포인트 초기값:100

카운터 증가

/카운터증가

유저의 카운터 값을 증가시켜요. 특정 유저의 카운터를 관리자가 증가시킬 수도 있어요!

/카운터증가 카운터이름:포인트 증가량:50 대상유저:@유저이름

카운터 확인

/카운터확인

유저의 카운터 값을 확인해요 특정 카운터만 확인하거나 모든 카운터를 확인할 수 있어요!

/카운터확인 카운터이름:포인트 대상유저:@유저이름

조건부 명령어 만들기

/조건명령만들기

조건에 따라 다른 응답을 제공하는 조건부 명령어를 생성해요 다양한 조건 유형을 사용할 수 있어요!

/조건명령만들기 명령어:환영 조건유형:역할 보유 조건값:@VIP 성공응답:VIP 회원님 환영합니다! 실패응답:일반 회원님 환영합니다!

랜덤 응답 명령어

/랜덤명령만들기

여러 응답 중 하나를 무작위로 선택하는 명령어를 생성해요 여러 줄로 구분하여 응답을 입력해요

/랜덤명령만들기 명령어:운세 응답목록:오늘은 행운이 가득한 날이에요!! 오늘은 조심해야 할 날입니다.. 오늘은 평범한 하루가 될 것 같아요!

쿨다운 설정

/쿨다운설정

명령어 사용에 쿨다운을 설정할수 있어요 명령어 남용을 방지하고 서버를 효율적으로 관리할 수 있어요

/쿨다운설정 명령어:운세 쿨다운시간:60

사용 방법

커스텀봇은 누구나 쉽게 코딩 없이 사용할 수 있습니다. 따라하기만 하면 금방 나만의 봇을 만들 수 있어요!

1

봇 초대하기

커스텀봇을 디스코드 서버에 초대하세요. 초대 버튼을 클릭하면 간단하게 초대할 수 있어요!

2

기본 설정하기

서버에 맞게 기본 설정을 변경해보세요. 명령어 접두사를 변경하거나 권한을 설정할 수 있어요!

3

명령어 만들기

다양한 종류의 명령어를 만들어보세요. 일반 텍스트 응답, 역할 부여, 임베드 등 원하는 대로 설정할 수 있어요!

4

고급 기능 활용하기

카운터, 타이머, 조건부 명령어 등 고급 기능을 활용하여 더 풍부한 서버 경험을 만들어보세요!

5

명령어 사용하기

설정한 접두사와 명령어 이름으로 간편하게 사용할 수 있습니다. 서버 멤버 누구나 쉽게 사용할 수 있어요!
